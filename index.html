Welcome to Workshop 3! In this session, we will explore how to control the flow of a JavaScript program using conditional statements. Specifically, we'll dive into if, else, and else if statements, as well as how to use logical operators to create complex conditions. Understanding these concepts will allow you to make your programs respond dynamically based on different conditions, which is essential for building interactive and functional applications.

Understanding Conditional Statements

Conditional statements allow your code to make decisions and execute specific blocks of code based on certain conditions. In JavaScript, the most basic form of a conditional statement is the if statement.

Comparison Operators

Comparison operators are used to compare two values. They return a Boolean value (true or false) based on the comparison.

Operator Description Example Result
== Equal to (loose equality, type coercion allowed) 5 == '5' true
=== Strict equal to (no type coercion) 5 === '5' false
!= Not equal to (loose inequality, type coercion allowed) 5 != '5' false
!== Strict not equal to (no type coercion) 5 !== '5' true
> Greater than 10 > 5 true
>= Greater than or equal to 10 >= 10 true
< Less than 5 < 10 true
<= Less than or equal to 5 <= 5 true

The if Statement

The if statement executes a block of code if a specified condition evaluates to true. 


if (condition) {
    // Code to run if the condition is true
}

Example: 


let age = 16;

if (age >= 16) {
    console.log("You are eligible to vote.");
}

In this example, the message "You are eligible to vote." will be printed to the console because the condition age >= 16 is true.

Adding Alternatives with else

The else statement allows you to specify an alternative block of code that runs if the condition in the if statement is false. 


if (condition1) {
    // Code to run if condition1 is true
} else if (condition2) {
    // Code to run if condition1 is false and condition2 is true
} else {
    // Code to run if both condition1 and condition2 are false
}

Example: 


let temperature = 75;

if (temperature > 90) {
    console.log("It's too hot!");
} else if (temperature > 60) {
    console.log("The weather is nice.");
} else {
    console.log("It's too cold!");
}

In this example, the message "The weather is nice." will be printed because the temperature is greater than 60 but not greater than 90.

Handling Multiple Conditions with else if

Sometimes, you may need to check multiple conditions. The else if statement allows you to specify a new condition to test if the previous condition(s) were false. 


if (condition1) {
    // Code to run if condition1 is true
} else if (condition2) {
    // Code to run if condition1 is false and condition2 is true
} else {
    // Code to run if both condition1 and condition2 are false
}

Example: 


let score = 85;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: F");
}

In this example, the message "Grade: B" will be printed because the score falls between 80 and 89.

Using Logical Operators

Logical operators allow you to combine multiple conditions in a single if, else if, or else statement. The most common logical operators are:
AND (&&): Returns true if both conditions are true.
OR (||): Returns true if at least one condition is true.
NOT (!): Inverts the truthiness of a condition.

AND Operator Example 


let age = 25;
let hasLicense = true;

if (age >= 18 && hasLicense) {
    console.log("You can drive.");
} else {
    console.log("You cannot drive.");
}

In this example, the message "You can drive." will be printed because both conditions are true.

OR Operator Example 


let isWeekend = true;
let isHoliday = false;

if (isWeekend || isHoliday) {
    console.log("You can relax today.");
} else {
    console.log("It's a workday.");
}

Here, the message "You can relax today." will be printed because isWeekend is true, even though isHoliday is false.

OR Operator Example 


let isRaining = false;

if (!isRaining) {
    console.log("You can go for a walk.");
} else {
    console.log("Better stay inside.");
}

The message "You can go for a walk." will be printed because !isRaining evaluates to true.

Q&A and Troubleshooting

  • ✅ Make sure you don't confuse assignment operator = with comparison operator ==.
  • ✅ If your if statement does not work, check your semi-colons. if(condition) does not use semi-colons.
  • 🦆 Logical Operators can be tricky, and frustrating if they don't do what you want. Read them aloud and you will hear the issue.
